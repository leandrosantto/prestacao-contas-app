3 Notinhas

Como adicionar várias de uma vez
Já escaneou? Escaneie no app Arquivos ou Notas (⋯ → Escanear Documentos). Ele salva um PDF com todas. Aqui, toque no botão → Escolher Arquivos → selecione o PDF: o app separa cada página numa nota.

Prefere fotos? Tire as fotos pela câmera e depois toque no botão → Biblioteca de fotos → dá pra marcar várias de uma vez.

Obs.: o iPhone não deixa sites abrirem o scanner direto — por isso o atalho é escanear no Arquivos/Notas e trazer o PDF pra cá.
